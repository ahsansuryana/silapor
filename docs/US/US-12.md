| Aspek | Detail |
|-------|--------|

| **Aktor** | Mahasiswa, Staff, Admin |

| **FR** | FR-14 |

## Story
> Sebagai **pengguna**, saya ingin **mendapatkan notifikasi saat ada perubahan pada laporan** agar **tidak perlu refresh terus menerus**.

## Acceptance Criteria

1. **Terima notifikasi**
   - Mahasiswa: dapat notifikasi saat status laporannya berubah
   - Staff: dapat notifikasi saat ada laporan baru di-assign
   - Admin: dapat notifikasi relevan

2. **Lihat notifikasi**
   - Halaman notifikasi khusus
   - Daftar notifikasi: title, body, waktu, status baca
   - Klik notifikasi → buka laporan terkait

3. **Unread count**
   - Badge jumlah notifikasi belum dibaca
   - Update real-time

4. **Mark as read**
   - Tandai satu notifikasi sebagai dibaca
   - Tandai semua sebagai dibaca
