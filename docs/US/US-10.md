## Story
> Sebagai **Admin**, saya ingin **melihat dashboard berisi statistik dan overview laporan** agar **bisa memantau kinerja sistem secara keseluruhan**.

## Acceptance Criteria

1. **Statistik overview**
   - Total laporan
   - Jumlah laporan per status (chart/donut)

2. **Trend mingguan**
   - Data laporan 7 hari terakhir
   - Perbandingan dengan minggu sebelumnya
   - Persentase kenaikan/penurunan

3. **Report center**
   - Semua laporan (tanpa filter user)
   - Bisa filter, cari, sorting
   - Bisa ubah status & assign manual

4. **Akses penuh**
   - Admin bisa lihat detail laporan manapun
   - Admin bisa update status laporan manapun
