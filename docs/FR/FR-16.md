# FR-16: Manajemen Pengguna

| Aspek | Detail |
|-------|--------|
| **ID** | FR-16 |
| **Nama** | Manajemen Pengguna |
| **Aktor** | Admin |
| **Prioritas** | Sedang |
| ** terkait** | UC-27, UC-28 |
| **Dependensi** | FR-1, FR-3 |

## Deskripsi
Admin dapat mengelola data pengguna (staff, mahasiswa) dan role.

## Staff Management (FR-24)
- `GET /api/staff` — daftar semua staff
- `PUT /api/staff/:id` — update data staff
- `DELETE /api/staff/:id` — hapus staff

## User Management (Mahasiswa)
- `GET /api/users` — daftar user (mahasiswa)
- `POST /api/users` — create user
- `PUT /api/users/:id` — update user
- `DELETE /api/users/:id` — delete user

## Frontend Pages
- **StaffManagement.tsx** — CRUD staff + assign lokasi
- **UserManagement.tsx** — CRUD mahasiswa
- **FacilityManagement.tsx** — manage staff-location mapping

## Update Role
`POST /api/auth/update-role` — admin dapat mengubah role user
