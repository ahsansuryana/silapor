## Deskripsi
Dashboard menampilkan ringkasan data dan statistik laporan.

## Admin Dashboard (AdminOverview)
- Total laporan
- Breakdown per status (count)
- Data mingguan (chart)
- Persentase perubahan (trend)
- Statistik per kategori

## Staff Dashboard (StaffDashboard)
- Total task aktif
- Task per status
- Task terbaru

## Endpoint
`GET /api/reports/stats` — statistik umum & admin dashboard

## Frontend
- **AdminOverview.tsx**: Statistik chart, cards
- **StaffDashboard.tsx**: Task overview, quick actions
