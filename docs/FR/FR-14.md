# FR-14: Notifikasi In-App

| Aspek | Detail |
|-------|--------|
| **ID** | FR-14 |
| **Nama** | Notifikasi In-App |
| **Aktor** | Mahasiswa, Staff, Admin |
| **Prioritas** | Sedang |
| ** terkait** | UC-23, UC-24 |
| **Dependensi** | FR-8, FR-11 |

## Deskripsi
Notifikasi dikirim ke pengguna saat terjadi event penting terkait laporan mereka.

## Trigger Notifikasi
| Event | Penerima | Title |
|-------|----------|-------|
| Laporan dibuat | Staff yang di-assign | "Laporan Baru" |
| Status berubah | Mahasiswa (reporter) | "Status Diupdate" |
| Assignment manual | Staff yang di-assign | "Tugas Baru" |
| Transfer | Staff baru | "Transfer Tugas" |

## Data Notifikasi
- `user_id` — penerima
- `report_id` — laporan terkait
- `title` — judul
- `body` — isi
- `is_read` — status baca
- `sent_at` — waktu dikirim

## Endpoints
- `GET /api/notifications` — daftar notifikasi user
- `GET /api/notifications/unread-count` — jumlah belum dibaca
- `PATCH /api/notifications/:id/read` — tandai baca
- `PATCH /api/notifications/read-all` — tandai semua baca
- `DELETE /api/notifications/:id` — hapus
