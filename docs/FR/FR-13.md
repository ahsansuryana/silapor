# FR-13: Manajemen Kategori

| Aspek | Detail |
|-------|--------|
| **ID** | FR-13 |
| **Nama** | Manajemen Kategori |

| ** terkait** | UC-22 |
| **Dependensi** | - |

## Deskripsi
Kategori digunakan untuk mengelompokkan jenis kerusakan yang dilaporkan.

## Data Kategori
- `name` (varchar 255) — Nama kategori
- `short_description` (text) — Deskripsi singkat

## Endpoints
- `GET /api/categories` — semua kategori (public)
- `GET /api/categories/:id` — detail kategori
- `POST /api/categories` — create (admin)
- `PATCH /api/categories/:id` — update (admin)
- `DELETE /api/categories/:id` — delete (admin)
