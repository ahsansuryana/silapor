# FR-10: Komentar Laporan

| Aspek | Detail |
|-------|--------|
| **ID** | FR-10 |
| **Nama** | Komentar Laporan |
| **Aktor** | Mahasiswa, Staff, Admin |
| **Prioritas** | Sedang |
| ** terkait** | UC-15 |
| **Dependensi** | FR-4 |

## Deskripsi
Pengguna dapat berdiskusi pada laporan melalui fitur komentar.

## Fitur
- Tambah komentar (message text)
- Lihat semua komentar (user_name, user_role, created_at)
- Tidak ada edit/hapus komentar (v1)

## Endpoint
- `GET /api/reports/:id/comments`
- `POST /api/reports/:id/comments`

## Input
```json
{
  "message": "Komentar teks (required)"
}
```
