# FR-11: Assignment & Transfer Staff

| Aspek | Detail |
|-------|--------|
| **ID** | FR-11 |
| **Nama** | Assignment & Transfer Staff |
| **Aktor** | Sistem (auto), Admin (manual), Staff (transfer) |
| **Prioritas** | Tinggi |
| ** terkait** | UC-16, UC-17, UC-18 |
| **Dependensi** | FR-4, FR-12 |

## Deskripsi
Laporan yang masuk akan di-assign ke staff secara otomatis berdasarkan lokasi. Admin dapat melakukan assign manual atau transfer ke staff lain.

## Auto-Assign Algorithm
```
function findStaffInHierarchy(locationId):
    staff = cari staff di user_staff_location WHERE location_id = locationId
    if staff ditemukan:
        return staff
    if location.parent_id != null:
        return findStaffInHierarchy(location.parent_id)  // naik ke parent
    return null  // tidak ada staff
```

## Endpoints
- `GET /api/assignments/my-tasks` — task staff yang login
- `GET /api/assignments/:reportId` — assignment suatu report
- `GET /api/assignments/:reportId/active` — assignment aktif
- `POST /api/assignments` — manual assign (admin)
- `POST /api/assignments/transfer` — transfer (admin/staff)
- `POST /api/assignments/auto-assign` — auto-assign ulang

## Data Tersimpan
- `report_id`, `assigned_to`, `assigned_by`
- `is_active` (boolean), `is_auto_assign` (boolean)
- `notes`, `created_at`
